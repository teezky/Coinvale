# Coinvale Removed Or Dropped

Removed or dropped solutions:
- `House` was removed as a separate building.
- `Town Hall` took over the old population-building role.
- the old multi-building logic was removed:
  - the game no longer builds many copies of the same building
  - the new direction is `1 building type / levels / worker slider`
- the separate `Production Breakdown` card was removed.
- the sidebar `Village Summary` was removed.
- the top `Village Snapshot` was removed.
- `Settlement Focus` was removed.
- the separate top `Net Production` block was removed.
- the older, more linear `Knowledge` / `Expansion` research list was replaced with a branch-based prerequisite tree.
- the old `+ / -` worker assignment UI was replaced with sliders.
- `Clay` was removed as a planned resource.

Changes that intentionally replaced an earlier direction:
- building upgrades are no longer based on a shared type-wide level
- `Workshop` no longer gives a vague `all village output` buff
  - the bonus now applies to baseline building output

Why this file exists:
- keep track of ideas that were already rejected or removed
- avoid accidentally bringing back discarded systems
- make the project’s decision history easier to follow
